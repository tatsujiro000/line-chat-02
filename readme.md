# 課題4 -LINE風アプリ（firebase利用)-
## ①課題内容（どんな作品か）
- LINE　Businessで応答モードとチャットモードを並行して使えないという課題があったので、それを解消できるツールを作ろうと思いました。

## ②工夫した点・こだわった点
- firebaseでフロントからバックまでまかなえる構成を考えてデプロイしたことです。
- expressでルーティング、テンプレートエンジンでviewsを作るなど、下地になるものを準備しました。

## ③質問・疑問（あれば）
